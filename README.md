# This is a weather app that tells weather of the desired city made using ReactJS.

### Main Highlights:

-   Fully responsive website (Across all devices)
-   You can search any city of the world to get the details of the weather like min and max temperature (in celcius and Fahreinheit), Wind speed, Air pressure and Humidity.

### Tech Used:-
-   HTML
-   CSS
-   ReactJS
